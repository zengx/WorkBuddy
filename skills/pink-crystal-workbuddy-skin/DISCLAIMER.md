# 资产版权与个性化权责声明（Personalization & Asset Disclaimer）

## 1. 本 skill 的性质：通用个性化换装服务

本 skill 是一套**通用个性化定制（换装）服务框架**，用于为 WorkBuddy 桌面客户端注入视觉主题。
它**不指定、不绑定任何特定图片或视频作为唯一 / 强制素材**。

随附仓库的示例资产仅为**演示（demo）用途**，用于验证注入链路，不代表本服务对任何素材的偏好或限定。包括：

- **示例背景**：「星空」动态版（animated WebP）/ 静态版（JPEG 单帧），当前默认主题所用；
- **示例源素材**：`assets/source/` 内原始 MP4——`shanhaijing_demo.mp4`（山海经动画）、`stars_demo.mp4`（星空动画），
  供使用者自行重生成 WebP/JPEG 或参考其生成参数，无需重新调用视频生成服务。

## 2. 企业 / 个人使用指引

- **企业用户**：可基于自身**企业文化、品牌 LOGO、品牌主色**及**已获合法授权的图片**，
  派生「企业需求版本」——将主题强调色 `--wb-accent`（默认 `#ff4d9c`）改为企业品牌主色，
  并替换 `assets/{mode}/bg.b64.txt` 中的背景资产即可。企业版分发前请确保所用素材已获授权。
- **个人用户**：可根据**个人喜好**，指定自己**拥有或已获授权**的任意图片 / 视频作为皮肤背景。
- **替换方法**：参见 `SKILL.md`「背景资产」章节（视频→WebP / 图片→JPEG，转 base64 写入后重建部署）。
