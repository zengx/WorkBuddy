# csharp-lsp

C# Language Server，为 CodeBuddy 提供代码智能和诊断功能。

## 支持的文件类型

`.cs`

## 安装方式

### 通过 .NET 工具（推荐）
```bash
dotnet tool install --global csharp-ls
```

### 通过 Homebrew (macOS)
```bash
brew install csharp-ls
```

## 系统要求

- .NET SDK 6.0 或更高版本
