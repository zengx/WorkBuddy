# /weixin-minigame-helper:publish — 发布小游戏

## 执行步骤

### 1. 解析参数

- 从用户输入中提取版本号（如 "1.0.0"）
- 如果用户没有提供版本号，询问用户需要发布的版本号
- 版本号格式应为 semver（X.Y.Z）
- 可选提取版本描述（--desc 后的内容，或用户自然语言中的描述）

### 2. 确定游戏目录

- 搜索当前工作目录中的 `game.js`
- 路径转为绝对路径

### 3. 确认发布

- 向用户确认：
  - 版本号
  - 版本描述（如有）
  - 游戏目录路径
- 告知用户："即将发布 v{版本号} 到微信平台，确认继续？"
- **注意：发布是重要操作，必须得到用户确认后才能执行**

### 4. 执行发布

- 调用 MCP 工具 `publish`，传入：
  - `workspacePath`: 游戏目录绝对路径
  - `version`: 版本号
  - `desc`: 版本描述（可选）

### 5. 处理结果

**成功：**
- 告诉用户：
  - "🎉 小游戏 v{版本号} 发布成功！"
  - "新版本已上传到微信平台，请登录微信公众平台 (mp.weixin.qq.com) 提交审核。"

**配置缺失：**
- 同 `device-test` 命令的配置引导逻辑

**其他错误：**
- 显示错误详情
- 常见问题提示：
  - 密钥过期 → 重新生成
  - 版本号已存在 → 使用更高版本号
  - IP 白名单 → 在公众平台添加 IP
