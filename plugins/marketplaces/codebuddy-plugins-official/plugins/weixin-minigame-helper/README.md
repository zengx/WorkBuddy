# WeiXin Mini Game Helper

微信小游戏 AI 开发助手 —— 预览、调试、真机测试、发布一站式解决。

## ✨ 功能概览

- 🎮 **浏览器预览**：本地实时运行微信小游戏，支持多机型模拟
- 🔄 **自动重载**：代码修改后自动刷新预览
- 📱 **真机测试**：一键生成二维码，微信扫码即可真机体验
- 🚀 **发布上传**：直接上传到微信平台，无需打开开发者工具
- 🤖 **AI 驱动**：代码生成后自动预览，发现错误自动修复

## 🚀 快速上手

安装插件后，AI 会自动获得小游戏开发能力，无需额外配置。

### 自然语言交互

直接告诉 AI 你想做什么：

- "帮我做一个打砖块小游戏"
- "预览一下小游戏"
- "真机测试"
- "发布到微信"

AI 会自动调用对应工具完成操作。

### 斜杠命令

| 命令 | 功能 |
|------|------|
| `/weixin-minigame-helper:preview` | 启动游戏预览 |
| `/weixin-minigame-helper:device-test` | 真机测试 |
| `/weixin-minigame-helper:publish 1.0.0` | 发布到微信 |

## 📱 真机测试 & 发布配置

真机测试和发布需要配置微信 AppID 和上传密钥：

1. 在 `project.config.json` 中确保 `appid` 字段已填写
2. 设置上传密钥（从[微信公众平台](https://mp.weixin.qq.com/) → 开发 → 开发设置 → 小程序代码上传密钥 下载）
3. 如遇 IP 白名单错误，请在微信公众平台 → 开发 → 开发设置 → IP白名单 中添加本机 IP

## 📋 项目结构要求

确保你的微信小游戏项目包含以下文件：

```
├── game.js                 # 入口文件（必须）
├── game.json               # 游戏配置（必须）
└── project.config.json     # 项目配置（必须）
```

## ❓ 常见问题

**Q: 提示"这不是一个标准的微信小游戏结构"**
A: 请检查项目是否包含 `game.js`、`game.json`、`project.config.json` 三个文件。

**Q: 真机预览提示配置缺失**
A: 请确保 `project.config.json` 中填写了正确的 `appid`，并配置了上传密钥。

**Q: 上传提示 IP 白名单错误**
A: 前往微信公众平台 → 开发 → 开发设置 → IP白名单，添加本机公网 IP。
