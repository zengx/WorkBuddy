# rust-analyzer-lsp

Rust Language Server，为 CodeBuddy 提供代码智能和分析功能。

## 支持的文件类型

`.rs`

## 安装方式

### 通过 rustup（推荐）
```bash
rustup component add rust-analyzer
```

### 通过 Homebrew (macOS)
```bash
brew install rust-analyzer
```

### 通过包管理器 (Linux)
```bash
# Ubuntu/Debian
sudo apt install rust-analyzer

# Arch Linux
sudo pacman -S rust-analyzer
```

### 手动下载
从 [发布页面](https://github.com/rust-lang/rust-analyzer/releases) 下载预编译二进制文件。
