# PPT Writer - AI驱动的演示文稿创作助手

一个功能强大的CodeBuddy Code插件，帮助你快速创建专业、美观、有说服力的PPT演示文稿。

## ✨ 核心功能

### 🎯 智能内容生成
- **自动生成大纲**：根据主题智能生成完整的PPT结构
- **AI撰写内容**：为每一页生成标题、要点和演讲备注
- **多种内容模板**：产品介绍、项目汇报、培训课程、商业计划等
- **案例和数据支撑**：自动建议相关案例和数据来源

### 🎨 专业视觉设计
- **多种模板风格**：商务、技术、教育、极简等
- **配色方案建议**：根据场景推荐专业配色
- **字体搭配指导**：提供字体选择和层次建议
- **图表类型推荐**：根据数据类型选择最佳可视化方式

### 📊 多格式输出
- **Markdown格式**：便于编辑和协作
- **JSON格式**：方便程序化处理
- **HTML格式**：可在浏览器中预览

### 🤖 AI助手支持
- **PPT设计专家Agent**：专业的设计顾问，提供全方位指导
- **自动激活技能**：根据需求自动调用设计和内容生成能力

### 基础使用

#### 1. 创建一个简单的PPT

```bash
/create-ppt "人工智能发展趋势"
```

这将生成一个包含10页的PPT大纲，使用默认模板。

#### 2. 自定义PPT参数

```bash
/create-ppt "企业数字化转型" --slides 15 --template business --format markdown
```

参数说明：
- `--slides`: 幻灯片数量（默认10）
- `--template`: 模板类型（business/tech/education/minimal）
- `--format`: 输出格式（markdown/json/html）

#### 3. 使用PPT设计专家

如果你需要更深入的设计指导，可以直接与PPT设计专家对话：

```
我需要为公司年会准备一个产品发布PPT，时长20分钟，受众是客户和合作伙伴
```

Agent会自动激活并提供专业建议。

## 📖 模板类型详解

### 商务模板 (business)
**特点**：专业、简洁、数据驱动
- **配色**：深蓝、金色、白色
- **字体**：Montserrat + Open Sans
- **适用场景**：董事会汇报、投资路演、年度总结

### 技术模板 (tech)
**特点**：现代、创新、视觉化
- **配色**：科技蓝、深灰、荧光绿
- **字体**：Roboto + Source Code Pro
- **适用场景**：技术分享、产品发布、架构设计

### 教育模板 (education)
**特点**：友好、清晰、循序渐进
- **配色**：暖橙、天蓝、米白
- **字体**：Poppins + Nunito
- **适用场景**：培训课程、知识分享、教学演示

### 极简模板 (minimal)
**特点**：简约、大字号、留白充足
- **配色**：黑色、白色
- **字体**：Helvetica Bold
- **适用场景**：TED式演讲、创意展示、品牌发布

## 🎓 使用技巧

### 内容创作最佳实践

1. **明确目标**：在创建前先思考演讲的核心目标
2. **了解受众**：根据听众背景调整内容深度
3. **控制信息量**：每页3-5个要点，避免信息过载
4. **数据说话**：用具体数据和案例增强说服力
5. **故事化表达**：用故事连接数据和观点

### 视觉设计最佳实践

1. **配色简洁**：最多3种主色
2. **字体统一**：不超过2-3种字体
3. **对比明显**：确保文字与背景对比度充足
4. **留白充足**：避免页面拥挤
5. **视觉层次**：通过大小、粗细、颜色建立层次

### 演讲准备技巧

1. **控制时长**：按照1-2分钟/页估算总时长
2. **重点演练**：对关键页面进行多次练习
3. **准备Q&A**：预想可能的问题并准备答案
4. **备用素材**：准备额外的数据和案例以备不时之需
5. **技术检查**：提前测试投影设备和文件兼容性

## ❓ 常见问题

### Q: 生成的PPT可以直接用于演示吗？
A: 插件生成的是PPT的内容大纲和设计建议，采用Markdown等格式。你需要将内容导入到PowerPoint、Keynote或Google Slides等工具中制作最终的演示文稿。

### Q: 如何选择合适的模板？
A: 根据你的演讲场景和受众选择：
- 正式商务场合 → business模板
- 技术分享或产品发布 → tech模板
- 培训教学 → education模板
- 创意展示或品牌发布 → minimal模板

### Q: 可以修改生成的内容吗？
A: 当然可以！生成的内容只是建议，你可以根据实际需求修改、增删或重组。

### Q: 支持中英文混合吗？
A: 支持。插件会根据输入的主题语言生成对应语言的内容。

### Q: 生成的图表建议如何实现？
A: 插件会建议图表类型和数据展示方式，你需要在PowerPoint等工具中使用相应的图表功能制作。
