# PPT视觉设计技能

自动为PPT演示提供专业的视觉设计建议和优化方案。

## 激活场景

此技能在以下情况自动激活：

1. 用户提到PPT配色、字体、布局等视觉相关词汇
2. 用户要求优化或美化现有PPT
3. 用户询问PPT设计规范和最佳实践
4. 用户需要选择合适的图表类型
5. 用户寻求视觉风格建议

## 核心能力

### 1. 配色方案设计

为不同场景提供专业配色：

**商务场景**：
```
主色：海军蓝 #0A2463
辅色：金色 #D4AF37, 浅灰 #E8E8E8
强调：深红 #C1292E
背景：白色 #FFFFFF
```

**科技场景**：
```
主色：科技蓝 #1E90FF
辅色：深灰 #2C3E50, 荧光绿 #00FF7F
强调：紫色 #8B00FF
背景：深色 #1A1A1A 或白色 #FFFFFF
```

**教育场景**：
```
主色：暖橙 #FF6B35
辅色：天蓝 #4ECDC4, 柠檬黄 #FFE66D
强调：深蓝 #293241
背景：米白 #F7FFF7
```

**极简场景**：
```
主色：黑色 #000000
辅色：白色 #FFFFFF
强调：单一品牌色
背景：白色或浅灰
```

### 2. 字体搭配建议

#### 标题字体（推荐）
- **Montserrat**：现代、几何、适合商务
- **Roboto**：清晰、中性、适合技术
- **Poppins**：友好、圆润、适合教育
- **Helvetica**：经典、简洁、适合极简

#### 正文字体（推荐）
- **Open Sans**：易读、专业
- **Lato**：优雅、现代
- **Nunito**：友好、平易
- **Source Sans Pro**：技术、清晰

#### 字体使用规则
```
标题：28-48号，粗体
副标题：20-28号，中等
正文：18-24号，常规
注释：14-16号，细体

最小字体：不小于14号（确保投影可读）
字体数量：最多2-3种字体
对比度：确保文字与背景对比度 ≥ 4.5:1
```

### 3. 布局网格系统

#### 标准布局模板

**标题页**：
```
┌─────────────────────────────┐
│                             │
│         [大标题]            │
│         [副标题]            │
│                             │
│        [作者/日期]          │
│                             │
└─────────────────────────────┘
```

**单栏内容**：
```
┌─────────────────────────────┐
│ [标题]                      │
│ ─────────────────────────── │
│ • 要点1                     │
│ • 要点2                     │
│ • 要点3                     │
│                             │
│        [图表/图片]          │
└─────────────────────────────┘
```

**双栏内容**：
```
┌─────────────────────────────┐
│ [标题]                      │
│ ─────────────────────────── │
│ • 要点1      │  [图片]      │
│ • 要点2      │   或         │
│ • 要点3      │  [图表]      │
│              │              │
└─────────────────────────────┘
```

**图表页**：
```
┌─────────────────────────────┐
│ [标题]                      │
│ ─────────────────────────── │
│                             │
│      [大型图表/图片]        │
│                             │
│    [数据说明或要点]         │
└─────────────────────────────┘
```

#### 留白原则
- 页边距：至少10%（左右各5%，上下各5%）
- 元素间距：标题下方留20-30px
- 行间距：1.2-1.5倍
- 避免填满整个页面

### 4. 图表类型选择

根据数据类型选择合适的图表：

**比较数据**：
- 柱状图：比较不同类别的数值
- 条形图：比较项目较多时使用
- 雷达图：多维度比较

**趋势数据**：
- 折线图：显示随时间变化趋势
- 面积图：强调变化幅度
- 瀑布图：显示增减变化

**占比数据**：
- 饼图：显示整体占比（≤5个分类）
- 环形图：突出重点占比
- 树状图：层级占比关系

**关系数据**：
- 散点图：显示相关性
- 气泡图：三维关系展示
- 热力图：矩阵关系

**流程数据**：
- 流程图：步骤和决策
- 时间轴：事件顺序
- 泳道图：跨部门流程

**层级数据**：
- 组织架构图：层级关系
- 金字塔图：层次结构
- 思维导图：知识体系

### 5. 视觉层次优化

#### 建立清晰层次的方法

**大小对比**：
```
标题：48号
副标题：28号
正文：20号
注释：14号
```

**粗细对比**：
```
标题：Bold (700)
副标题：SemiBold (600)
正文：Regular (400)
注释：Light (300)
```

**颜色对比**：
```
主要信息：深色（黑色/深蓝）
次要信息：中灰色
辅助信息：浅灰色
强调信息：品牌色/强调色
```

**位置层次**：
```
重要信息：页面上部、中心位置
次要信息：页面下部、边缘位置
```

### 6. 图标和图片使用

#### 图标选择
- **风格统一**：使用同一套图标库
- **大小一致**：相同层级图标大小相同
- **颜色协调**：与配色方案匹配
- **语义明确**：图标含义清晰

推荐图标库：
- Lucide Icons（开源、现代）
- Heroicons（简洁、清晰）
- Font Awesome（丰富、经典）
- Material Icons（标准、通用）

#### 图片使用
- **高质量**：分辨率 ≥ 1920x1080
- **相关性**：与内容直接相关
- **一致性**：风格和色调统一
- **版权**：使用正版或免费商用图片

推荐图片来源：
- Unsplash（高质量免费图片）
- Pexels（商用免费视频图片）
- Pixabay（多语言免费素材）

## 常见设计问题和解决方案

### 问题1：信息过载
**表现**：每页文字太多，密密麻麻
**解决**：
- 提炼核心观点，删减冗余信息
- 拆分成多页，一页一个重点
- 使用图表替代大段文字
- 增加留白空间

### 问题2：配色混乱
**表现**：颜色太多，缺乏主次
**解决**：
- 确定1-2个主色
- 使用同色系深浅变化
- 强调色仅用于重点内容
- 参考成熟的配色方案

### 问题3：字体杂乱
**表现**：多种字体混用，风格不统一
**解决**：
- 限制字体种类（最多3种）
- 标题用一种，正文用一种
- 通过大小和粗细建立层次
- 避免使用艺术字体

### 问题4：排版拥挤
**表现**：元素贴边，缺少呼吸感
**解决**：
- 设置合理的页边距
- 增加元素之间的间距
- 减少单页内容量
- 使用网格系统对齐

### 问题5：图表不清晰
**表现**：图表复杂，难以理解
**解决**：
- 简化数据，只展示关键信息
- 选择更直观的图表类型
- 添加清晰的标注和说明
- 使用对比色突出重点

## 快速设计技巧

### 5分钟快速美化
1. 统一字体（选1个标题字体+1个正文字体）
2. 应用配色方案（选1个主色+1个辅色）
3. 对齐所有元素（使用对齐工具）
4. 增加留白（调整页边距和间距）
5. 统一图标风格（替换为同一套图标）

### 配色快速选择
- 蓝色系：专业、可信、科技
- 绿色系：健康、环保、成长
- 橙色系：活力、创意、温暖
- 紫色系：高端、创新、神秘
- 灰色系：简约、现代、中性

### 应急设计方案
当时间紧迫时：
1. 使用纯色背景+白色文字（或反之）
2. 只用一种字体，通过大小区分层次
3. 使用简单的柱状图/折线图
4. 最小化装饰，聚焦内容
5. 确保对比度和可读性
