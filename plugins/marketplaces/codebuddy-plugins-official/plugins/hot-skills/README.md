# Hot Skills

A curated collection of 7 top-downloaded AI agent skills, bundled into a single plugin for easy installation.

## Included Skills

| Skill | Description |
|-------|-------------|
| **find-skills** | Discover and install agent skills from the open ecosystem |
| **summarize** | Summarize URLs, files, and YouTube links via the summarize CLI |
| **proactive-agent** | Transform agents from task-followers into proactive partners |
| **obsidian** | Work with Obsidian vaults and automate via obsidian-cli |
| **humanizer** | Remove signs of AI-generated writing from text |
| **self-improving** | Self-reflection, self-criticism, and self-learning memory system |
| **skill-vetter** | Security-first skill vetting before installing unknown code |

## Installation

Install via CodeBuddy marketplace:

```
/install hot-skills
```

## Skill Details

### find-skills
Helps users discover and install skills when they ask "how do I do X", "find a skill for X", or express interest in extending capabilities.

### summarize
Fast CLI to summarize URLs, local files, and YouTube links. Requires the `summarize` CLI tool.

### proactive-agent
A proactive, self-improving architecture for AI agents. Includes WAL Protocol, Working Buffer, Autonomous Crons, security hardening, and battle-tested patterns.

### obsidian
Work with Obsidian vaults (plain Markdown notes) and automate via `obsidian-cli`. Requires the `obsidian-cli` tool.

### humanizer
Identifies and removes signs of AI-generated text based on Wikipedia's "Signs of AI writing" guide. Covers 24 patterns including inflated symbolism, promotional language, AI vocabulary, and more.

### self-improving
Agent evaluates its own work, catches mistakes, and improves permanently. Features tiered memory storage, automatic promotion/demotion, and self-reflection logging.

### skill-vetter
Security-first vetting protocol for AI agent skills. Checks for red flags, permission scope, and suspicious patterns before installing any skill.
