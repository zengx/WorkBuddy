# swift-lsp

Swift Language Server (SourceKit-LSP)，为 CodeBuddy 提供 Swift 项目的代码智能功能。

## 支持的文件类型

`.swift`

## 安装方式

SourceKit-LSP 已包含在 Swift 工具链中。

### macOS
从 App Store 安装 Xcode，或通过以下方式安装 Swift：
```bash
brew install swift
```

### Linux
从 [swift.org](https://www.swift.org/download/) 下载并安装 Swift。

安装完成后，`sourcekit-lsp` 应该已经在 PATH 中可用。
