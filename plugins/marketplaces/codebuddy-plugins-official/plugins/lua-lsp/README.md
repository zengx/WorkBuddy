# lua-lsp

Lua Language Server，为 CodeBuddy 提供代码智能和诊断功能。

## 支持的文件类型

`.lua`

## 安装方式

### 通过 Homebrew (macOS)
```bash
brew install lua-language-server
```

### 通过包管理器 (Linux)
```bash
# Ubuntu/Debian (通过 snap)
sudo snap install lua-language-server --classic

# Arch Linux
sudo pacman -S lua-language-server

# Fedora
sudo dnf install lua-language-server
```

### 手动安装
从 [发布页面](https://github.com/LuaLS/lua-language-server/releases) 下载预编译二进制文件。
