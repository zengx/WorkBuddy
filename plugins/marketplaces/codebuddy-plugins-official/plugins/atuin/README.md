# Atuin 安全插件

自动拦截 AI 的高危操作，自动阻止 AI 使用有漏洞的组件。腾讯玄武实验室出品。

## 功能

- 自动监控包安装命令 (npm, pip, yarn, cargo 等)
- 检测有漏洞的依赖组件
- 拦截高危操作

## 命令

| 命令 | 描述 |
|------|------|
| `/atuin:code_review` | 代码审查 - 分析 git diff 并提供审查意见 |
