# pyright-lsp

Python Language Server (Pyright)，为 CodeBuddy 提供静态类型检查和代码智能功能。

## 支持的文件类型

`.py`, `.pyi`

## 安装方式

通过 npm 全局安装：

```bash
npm install -g pyright
```

或使用 pip：

```bash
pip install pyright
```

或使用 pipx（推荐用于 CLI 工具）：

```bash
pipx install pyright
```
