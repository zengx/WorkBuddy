# php-lsp

PHP Language Server (Intelephense)，为 CodeBuddy 提供代码智能和诊断功能。

## 支持的文件类型

`.php`

## 安装方式

通过 npm 全局安装：

```bash
npm install -g intelephense
```

或使用 yarn：

```bash
yarn global add intelephense
```
