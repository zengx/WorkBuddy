# gopls-lsp

Go Language Server，为 CodeBuddy 提供代码智能、重构和分析功能。

## 支持的文件类型

`.go`

## 安装方式

使用 Go 工具链安装：

```bash
go install golang.org/x/tools/gopls@latest
```

确保 `$GOPATH/bin`（或 `$HOME/go/bin`）已添加到 PATH 环境变量。
