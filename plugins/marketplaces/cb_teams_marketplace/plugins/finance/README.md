# Finance 财务与会计插件

财务与会计插件，支持月末结账、日记账分录、账户核对、财务报表生成、差异分析和 SOX 审计支持。

> **重要提示**：本插件协助财务和会计工作流程，但不提供财务、税务或审计建议。所有输出应在用于财务报告、监管申报或审计文档之前由合格的财务专业人员审核。

## 安装

```bash
# 通过 CodeBuddy 插件市场安装
/plugin install finance@cb-teams
```

## 功能说明

这个插件将 CodeBuddy 转变为财务与会计专家，提供系统化的财务工作流程和专业标准指导。

### 核心能力

#### 1. 财务工作流程（5种完整流程）
- ✅ 日记账分录准备（应付账款计提、固定资产、预付费用、工资、收入）
- ✅ 账户核对（总账对子账、银行对账、第三方对账）
- ✅ 损益表生成（期间对比和差异分析）
- ✅ 差异分析（驱动因素分解和瀑布图分析）
- ✅ SOX 合规测试（样本选择、测试工作底稿、控制评估）

#### 2. 月末结账支持
- 任务清单和依赖关系管理
- 日常结账活动序列
- 状态跟踪和进度监控
- 常见结账问题和解决方案

#### 3. 审计支持
- SOX 404 控制测试方法论
- 样本选择技术
- 文档标准和要求
- 缺陷分类和报告

## Skills

| Skill | 描述 |
|-------|------|
| **finance-workflows** | 5种完整的财务工作流程（日记账、核对、报表、差异、审计） |
| **journal-entry-prep** | 日记账分录最佳实践和支持文档要求 |
| **reconciliation** | 核对方法论（总账对子账、银行、跨公司） |
| **financial-statements** | 财务报表格式（损益表、资产负债表、现金流量表）和 GAAP 呈现 |
| **variance-analysis** | 差异分解技术（价格/数量、比率/组合）和叙述生成 |
| **close-management** | 月末结账清单、任务排序和状态跟踪 |
| **audit-support** | SOX 404 控制测试方法论和文档标准 |

## 使用示例

### 场景1：月末结账
```
User: 我需要为 2024年12月 准备月末结账分录

CodeBuddy使用finance插件:
1. 生成应付账款计提分录
2. 生成预付费用摊销分录
3. 生成固定资产折旧分录
4. 执行现金账户核对
5. 执行应收账款子账核对
6. 生成损益表和差异分析
```

### 场景2：账户核对
```
User: 帮我核对2024年12月的银行账户

CodeBuddy:
1. 收集总账现金余额和银行对账单余额
2. 识别在途存款和未兑现支票
3. 识别银行费用和利息
4. 生成核对工作底稿，列出所有调节项
5. 提供核对摘要和需要的调整分录
```

### 场景3：差异分析
```
User: 分析 2024年Q4 vs 2024年Q3 的收入差异

CodeBuddy:
1. 收集两个期间的收入数据
2. 将差异分解为驱动因素（价格、数量、产品组合等）
3. 生成瀑布图可视化差异驱动因素
4. 提供每个重大差异的叙述解释
5. 标记需要进一步调查的异常变化
```

### 场景4：日记账分录
```
User: 生成2024年12月的固定资产折旧分录

CodeBuddy:
1. 收集固定资产明细（成本、累计折旧、折旧方法）
2. 计算本期折旧费用
3. 生成包含借贷方的分录
4. 提供支持文档要求（折旧计算表）
5. 包含审核清单确保分录准确性
```

### 场景5：SOX 测试
```
User: 为2024年Q4准备收入确认控制测试

CodeBuddy:
1. 生成控制矩阵，列出关键控制点
2. 执行样本选择（统计或判断抽样）
3. 创建测试工作底稿模板
4. 提供测试步骤和预期结果
5. 指导缺陷分类（设计/运行有效性）
```

## 数据输入方式

此插件支持多种数据输入方式：

✅ **手动粘贴数据** - 直接粘贴试算表、总账余额、子账数据  
✅ **上传文件** - 支持 Excel、CSV 格式的财务数据  
✅ **提供模板** - 插件可生成数据收集模板供您填写  
✅ **描述结构** - 告诉数据结构，获得分析建议

## 最佳实践

### 日记账分录
- 使用标准会计科目表
- 包含充分的支持文档
- 添加清晰的分录说明
- 确保借贷平衡
- 遵循公司会计政策

### 账户核对
- 按月执行核对
- 及时调查未核对项
- 记录老化未核对项
- 保留核对工作底稿
- 获得适当的审核批准

### 财务报表
- 遵循 GAAP 呈现标准
- 包含期间对比分析
- 解释重大差异
- 标注非正常项目
- 验证计算准确性

### 差异分析
- 设置重要性门槛
- 分解为具体驱动因素
- 提供数据支持的解释
- 识别趋势和模式
- 标记需要行动的项目

### 审计支持
- 遵循 SOX 404 要求
- 使用适当的样本量
- 记录测试步骤和结果
- 保留完整的审计底稿
- 及时沟通发现的问题

- ✅ 内部审计：SOX 测试、控制评估、合规检查
- ✅ 财务经理：结账管理、团队协调、审核流程
- ✅ CFO/控制人员：财务监督、报告审核、政策合规
