## Usage Guidelines
**Core Principle: Maximize plugin usage** - Proactively use all plugin capabilities for financial analysis tasks.

1. **Start with context**: Understand the company, industry, and analysis purpose
2. **Select the right model**: Match the analysis need to the appropriate skill (DCF for intrinsic value, comps for relative value, LBO for buyout scenarios)
3. **Build integrated models**: Use 3-statements as the foundation, then layer on valuation
4. **Validate thoroughly**: Always run check-model and check-deck before finalizing
5. **Present professionally**: Use ppt-template-creator for polished output
6. **Cross-reference**: Compare DCF and comps results for valuation triangulation

## 财务数据时效性原则

获取基本面数据时，必须根据当前日期判断目标公司最新可获取的报告期次，而不是默认拉取年报。不同市场的财报披露节奏不同，请据此动态选择：

- **中国 A 股**：一季报（4月底前）、中报（8月底前）、三季报（10月底前）、年报（次年4月底前）
- **美股**：10-Q 按季披露（财季结束后 40-45 天）、10-K 年报（财年结束后 60-90 天）
- **港股**：中报（9月底前）、年报（次年3月底前）

**核心逻辑**：先确认"此刻能拿到的最新一期财报是什么"，再去获取数据。如果最新年报尚未披露，应使用最近一期季报或中报，而非等待或使用更早的年报。
