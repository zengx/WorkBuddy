# Data 数据分析插件

数据分析插件，支持 SQL 查询、数据探索、可视化、仪表板构建和洞察生成。包含完整的数据分析工作流程和最佳实践。

## 安装

```bash
# 通过 CodeBuddy 插件市场安装
/plugin install data@cb-teams
```

## 功能说明

这个插件将 CodeBuddy 转变为数据分析专家，提供系统化的数据分析工作流程和最佳实践指导。

### 核心能力

#### 1. 数据分析工作流程（6种完整流程）
- ✅ 回答数据问题（快速查询到完整报告）
- ✅ 探索和分析数据集
- ✅ 编写优化的 SQL 查询
- ✅ 创建专业可视化
- ✅ 构建交互式仪表板
- ✅ 分享前的分析验证

#### 2. SQL 查询专家
跨多种数据库方言编写优化查询：
- PostgreSQL / Aurora / Supabase
- Snowflake / BigQuery
- Databricks / Spark SQL
- Redshift / MySQL / MariaDB

#### 3. 数据探索与分析
- 数据分析和质量评估
- 模式发现和异常检测  
- 统计分析和假设检验
- 结果验证和QA检查

#### 4. 可视化与仪表板
- Python 可视化（matplotlib、seaborn、plotly）
- HTML/JS 交互式仪表板（Chart.js）
- 设计最佳实践和可访问性

## 使用示例

### 场景1：回答数据问题
```
User: 分析过去 12 个月的月度收入趋势，按产品线细分

CodeBuddy使用data插件: 
1. 理解问题（时间序列、产品维度、收入指标）
2. 生成SQL查询或要求提供数据
3. 计算月度趋势和同比增长
4. 创建趋势图表
5. 识别关键模式并验证结果
```

### 场景2：探索新数据集
```
User: 我有一份用户数据CSV，帮我分析

CodeBuddy: 
1. 读取并分析数据集结构
2. 生成数据质量报告（空值、异常值）
3. 推荐高价值分析维度
4. 建议下一步分析方向
```

### 场景3：编写SQL查询
```
User: 我需要一个队列留存分析查询，Snowflake数据库

CodeBuddy:
1. 理解需求（按注册月分组、计算留存率）
2. 使用Snowflake语法
3. 构建清晰的CTE查询
4. 添加注释和性能说明
```

### 场景4：创建可视化
```
User: 用这份销售数据创建趋势图

CodeBuddy:
1. 分析数据选择图表类型
2. 生成Python代码（matplotlib/seaborn）
3. 应用设计最佳实践
4. 输出高质量图表
```

### 场景5：构建仪表板
```
User: 创建销售仪表板，包含月度收入和区域细分

CodeBuddy:
1. 设计仪表板布局（KPI卡片、图表、过滤器）
2. 生成独立HTML文件（无需服务器）
3. 集成Chart.js可视化
4. 添加交互过滤器
```

### 场景6：验证分析
```
User: 检查这份分析报告

CodeBuddy:
1. 审查方法论和假设
2. 检查常见陷阱（偏差、逻辑错误）
3. 验证结果合理性
4. 评估置信度并标注局限性
```

## 数据输入方式

此插件支持多种数据输入方式：

✅ **手动粘贴数据** - 直接粘贴查询结果或数据表
✅ **上传文件** - 支持 CSV、Excel、JSON、Parquet  
✅ **生成SQL** - 编写查询供您在数据库中执行
✅ **描述结构** - 告诉结构，获得查询建议

## 最佳实践

### SQL查询
- 使用CTE提高可读性
- 明确指定JOIN类型
- 添加注释解释复杂逻辑
- 考虑性能优化

### 数据探索
- 先了解数据结构再分析
- 检查空值和异常值
- 关注数据质量问题

### 可视化
- 选择合适的图表类型
- 保持简洁清晰
- 使用无障碍配色
- 添加清晰标签

### 分析验证
- 进行合理性检查
- 警惕统计陷阱
- 记录假设和局限性
- 分享前验证结果
