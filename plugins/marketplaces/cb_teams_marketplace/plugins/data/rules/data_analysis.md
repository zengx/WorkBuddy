## Usage Guidelines

**Core Principle: Maximize plugin usage** - Actively use the data plugin's skills and workflows for all data analysis tasks.

1. **Start with understanding**: Clarify the user's data analysis needs
2. **Leverage plugin workflows**: Use the 6 complete workflows provided by the plugin
3. **Flexible data input**: Support manual paste, file upload, or SQL query generation
4. **Visualization first**: Use plotly for interactive charts whenever possible
5. **Create comprehensive reports**: When generating multiple visualizations, consolidate all charts into a complete report with narrative context
6. **Validate before presenting**: Always run validation checks on results
7. **Provide actionable insights**: Don't just show data, explain what it means

**Note**: This plugin works independently without requiring MCP server configuration. All data can be provided through manual input, file uploads, or by generating SQL queries for users to execute.
