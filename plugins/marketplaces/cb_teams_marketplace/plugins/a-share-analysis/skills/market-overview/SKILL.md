# 股票市场分析

---

## 适用边界

**使用本技能的情形：**

- 问句涉及全球或某市场(A股/港股/美股等)的整体行情或综述
- 涉及市场情绪、投资者信心、风险偏好判断
- 涉及龙虎榜数据解读、主力资金动向、北向资金
- 涉及板块轮动、市场热点主题、近期市场关注焦点

**不使用本技能的情形：**

- 问句聚焦某只具体股票的深度分析
- 问句聚焦宏观政策、国家经济形势
- 问句要求具体买卖建议或选股推荐

---

## 数据获取

本技能通过 finance-data plugin 获取所需的金融数据(行情、资金流向、龙虎榜、新股等)。

**获取原则：**
1. 根据分析主题，获取对应维度的市场数据
2. 如某项数据暂时无法获取，可调整查询维度或换用相近指标
3. 问句涉及多个维度时，分别获取各维度所需数据

---

## 第一步：确定分析主题

从问句、输入内容中通过理解，确定属于以下哪个分析主题(可多选，一般 1-2 个，不超过 3 个)：

| 分析主题     | 适用场景              |
| -------- | ----------------- |
| 全球股市联动综述 | 判断跨市场共振、风险传导与配置方向 |
| 全球金融市场综述 | 跨资产把握全球风险偏好与交易主线  |
| 盘前市场综述   | 开盘前把握重要事件、数据与关注点  |
| 盘中市场简评   | 盘中快速理解行情异动与驱动因素   |
| 盘后市场简报   | 收盘后复盘涨跌原因并梳理后市线索  |
| 新股情报     | 关注新股发行、申购与上市表现    |
| 再融资情报    | 关注再融资事件对资金流动的影响   |
| 每日大宗交易解读 | 关注大宗成交、折价溢价与机构动向  |
| 每日龙虎榜解读  | 跟踪短线热点、游资席位与交易行为  |
| 南向沪港通解读  | 关注南向资金流向与港股热点变化   |
| 市场情绪解读   | 用情绪指标判断风险偏好与拐点信号  |

---

## 第二步：获取数据

根据确定的分析主题，通过 finance-data plugin 获取对应的市场数据。

不同分析主题对应的数据获取方向：

| 分析主题 | 需要获取的数据类型 |
|---------|--------------|
| 全球股市联动综述 | 全球主要指数行情、跨市场涨跌对比 |
| 盘前/盘中/盘后综述 | A股指数行情、涨跌家数、成交额、板块涨跌 |
| 新股情报 | IPO新股上市数据、申购信息 |
| 再融资情报 | 再融资相关数据 |
| 每日大宗交易解读 | 大宗交易数据 |
| 每日龙虎榜解读 | 龙虎榜数据、机构交易明细 |
| 南向沪港通解读 | 沪深港通资金流向、港股通十大成交股 |
| 市场情绪解读 | 涨跌停数据、成交额、换手率等情绪指标 |

问句涉及多个维度时，分别获取各维度所需数据并行处理。

如某项数据暂时无法获取，可调整查询维度或换用相近指标。

---

## 第三步：输出

根据获取的数据，结合专业分析框架，针对问句、输入内容进行回答。
输出的内容应该以合适的统一的 markdown 形式给出。
输出字数与问题复杂度匹配：单一主题解读 300-500 字，多维市场综合分析 600-1000 字，除非用户特别要求，不超过 1500 字。

---

## 第四步：输出规范

**强制要求：**

- 分析内容应基于获取的数据和专业分析框架
- 引用机构观点时标注"据机构观点"，不注明具体来源
- 取不到的指标直接省略，不在正文中说明原因
- 正文结束即为输出结尾，不附加任何说明性文字

**严禁出现：**

- "受数据限制……"、"工具未返回……"
- "以上仅供参考"、"免责声明"、"几点说明"、"注："
- 任何解释工具调用状态的文字
