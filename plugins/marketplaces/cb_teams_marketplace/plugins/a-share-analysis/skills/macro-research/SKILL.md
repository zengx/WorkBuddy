# 宏观研究

---

## 适用边界

**使用本技能的情形：**

- 问句涉及某个国家、地区或全球整体的宏观经济状况
- 涉及货币政策、财政政策、利率、通胀、汇率、贸易政策
- 涉及地缘政治冲突对经济或金融市场的影响
- 涉及区域经济体、自贸区、特定经济政策的深度分析

**不使用本技能的情形：**

- 问句聚焦个股投资分析 
- 问句聚焦A股大盘今日行情、板块涨跌
- 问句要求推荐具体标的或配置方案

---

## 数据获取

本技能通过 finance-data plugin 获取所需的宏观经济数据(GDP、CPI、PPI、利率、汇率等)。

**获取原则：**
1. 根据分析主题和涉及的国家/地区，获取对应的宏观经济指标数据
2. 如某项数据暂时无法获取，可调整查询维度或换用相近指标
3. 问句涉及多个地区或维度时，分别获取各维度所需数据

---

## 第一步：确定分析主题

从问句、输入内容中通过理解，确定属于以下哪个分析主题(可多选，一般 1-2 个，不超过 3 个)：

| 分析主题   | 适用场景              |
| ------ | ----------------- |
| 全球宏观综述 | 把握全球增长、通胀与政策主线    |
| 宏观经济综述 | 中美欧等主要经济体数据与政策影响  |
| 国际关系解读 | 评估地缘事件对市场与行业的影响   |
| 财政政策解读 | 理解财政政策取向及对资产定价的影响 |
| 货币政策简报 | 跟踪利率、央行操作与流动性变化   |
| 区域经济分析 | 分析省市产业结构与区域增长潜力   |

---

## 第二步：获取数据

根据确定的分析主题和涉及的地区，通过 finance-data plugin 获取宏观经济数据。

**数据获取策略：**
- 全球维度：获取主要经济体的核心宏观指标(GDP、通胀、利率等)
- 特定国家/地区：聚焦该地区的关键经济数据和政策动态
- 多地区交叉分析：分别获取各地区数据后进行比较

不同分析主题对应的数据获取方向：

| 分析主题 | 需要获取的数据类型 |
|---------|--------------|
| 全球宏观综述 | 主要经济体GDP、通胀、PMI等指标 |
| 宏观经济综述 | 特定经济体核心数据与政策变化 |
| 国际关系解读 | 相关国家贸易数据、汇率、资本流动 |
| 财政政策解读 | 财政收支、国债收益率、政策文件 |
| 货币政策简报 | 利率数据(Shibor/LPR等)、货币供应量 |
| 区域经济分析 | 区域GDP、产业结构、投资数据 |

问句涉及多个地区或维度时，分别获取各维度所需数据并行处理。

如某项数据暂时无法获取，可调整查询维度或换用相近指标。

---

## 第三步：输出

根据获取的数据，结合专业分析框架，针对问句、输入内容进行回答。
输出的内容应该以合适的统一的 markdown 形式给出。
输出字数与问题复杂度匹配：单一经济体概述 300-500 字，多维或跨地区分析 600-1000 字，除非用户特别要求，不超过 1500 字。

---

## 第四步：输出规范

**强制要求：**

- 分析内容应基于获取的数据和专业分析框架
- 引用机构或政策表态时标注"据公开表态"，不注明具体来源机构
- 取不到的指标直接省略，不在正文中说明原因
- 正文结束即为输出结尾，不附加任何说明性文字

**严禁出现：**

- "受数据限制……"、"工具未返回……"
- "以上仅供参考"、"免责声明"、"几点说明"、"注："
- 任何解释工具调用状态的文字
