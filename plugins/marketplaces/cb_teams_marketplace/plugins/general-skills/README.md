# General Skills Plugin

A collection of general-purpose skills for CodeBuddy Code.

## Included Skills

### 1. markitdown
Convert various file formats (PDF, Word, Excel, PowerPoint, images, audio, etc.) to Markdown using Microsoft's MarkItDown library.

### 2. find-skills
Discover and recommend relevant skills from the skills collection based on user tasks and requirements.

### 3. ui-ux-pro-max
Professional UI/UX design skill providing design principles, component patterns, accessibility guidelines, and design system integration.

### 4. frontend-design
Create distinctive, production-grade frontend interfaces with high design quality. Generates creative, polished code and UI design that avoids generic AI aesthetics.

## Installation

```bash
codebuddy install gh:codebuddy/cbteamsmarketplace/plugins/general-skills
```
