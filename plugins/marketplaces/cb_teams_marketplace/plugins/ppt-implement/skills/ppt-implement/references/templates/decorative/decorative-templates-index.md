# PPT Content Page Template Index

## 📋 Usage Instructions
- Each template has a unique number, allocated in independent ranges by type
- All templates are split into individual `.tpl` files, organized by style in subfolders
- FilePath format: `number.tpl`
- Simply read the `.tpl` file from the corresponding style subfolder when using
