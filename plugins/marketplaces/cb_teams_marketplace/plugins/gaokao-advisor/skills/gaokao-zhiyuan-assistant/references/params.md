# 冲稳保参数表（params.md）

> 本文件定义阶段四「推荐流水线」与「冲稳保判定」的默认可配置参数。
> 所有阈值均为**初版默认值**，须按省份、批次、年份回测后校准。
> 判定结果一律为「基于近 X 年数据的参考判断，非录取保证」。

## 一、核心判定量：位次比 k

定义：

```
k = R_school / R_user
```

- `R_user`：考生的一分一段**位次**（数字越小越靠前）。
- `R_school`：目标**专业组/专业**的近年录取位次（**必须用专业组/专业位次，不能用院校最低位次**）。
- 取近 2-3 年位次的区间/趋势，不用单年点估；建议取近 3 年的中位或加权（近年权重更高）。

### k 区间 → 冲 / 稳 / 保

| 档位 | k 区间（默认） | 含义 |
| --- | --- | --- |
| 冲（chong） | `0.80 ≤ k < 0.95` | 院校位次高于我（更难），有希望但不稳 |
| 稳（wen） | `0.95 ≤ k ≤ 1.08` | 与我相当或我略优，大概率稳 |
| 保（bao） | `k > 1.08` | 明显低于我（更易），兜底 |
| 不建议（太冲） | `k < 0.80` | 风险过高，默认剔除（除非用户明确要冲） |

> 说明：k < 1 表示该校录取位次比考生更靠前（更难考），k > 1 表示更易。
> 边界值可按「可接受亏分程度」整体平移（见第三节）。

## 二、配比与候选数量

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| 冲∶稳∶保 配比 | `3 ∶ 4 ∶ 3` | 可配；保守型用户可调为 2∶4∶4 |
| 候选系数 | `1.5` | 候选总数 ≥ 本省可填志愿数 × 1.5 |
| 候选不足处理 | — | 不够则提示用户放宽城市/层次条件或补充心仪院校，**不自行降低数据标准** |

示例：本省可填 96 个志愿 → 候选池目标 ≥ 144 条；按 3∶4∶3 → 冲约 43 / 稳约 58 / 保约 43。

## 三、可接受亏分程度 → 调节「冲」的激进度

用户在阶段一/三给出的「可接受亏分」用于平移冲档的下界 `k_chong_low`：

| 可接受亏分倾向 | k_chong_low 调整 | 效果 |
| --- | --- | --- |
| 保守（不愿冒险） | `0.85` | 冲档收窄，更稳 |
| 适中（默认） | `0.80` | 默认 |
| 激进（愿意博） | `0.72` | 冲档放宽，可冲更高 |

> 仅调节「冲」的下界，稳/保区间不随之改变，避免兜底失效。

## 四、不确定性标注（每条判定必带）

每行冲稳保判定**必须**附带以下不确定性来源说明：

- 大小年波动（招生计划/报考热度的年度起伏）
- 招生计划变动（当年计划数与往年差异）
- 位次 ≠ 分数（同分位次随年份漂移）
- 专业组位次 vs 院校位次的区别（同校不同组差异大）
- 专业组「干净度」：组内若含用户明确拒绝的专业，标注调剂风险

统一话术：**「基于近 X 年数据的参考判断，非录取保证。」**

## 五、专业组干净度对服从调剂建议的影响

| 组内情况 | 服从调剂建议倾向 |
| --- | --- |
| 全部为用户可接受专业 | 建议「服从」（被调剂也能接受） |
| 含 1-2 个用户中性专业 | 「服从，但提示存在被调剂到 X 的可能」 |
| 含用户**明确拒绝**的专业 | 「谨慎/不建议服从」并标红风险 |

## 六、回测与校准待办

- [ ] 按省份分别校准 k 区间（不同省份位次密度不同）。
- [ ] 区分批次（本科批/专项/提前批）单独设阈值。
- [ ] 用近 3 年实际录取数据回测默认 k 区间的命中率，迭代边界值。
- [ ] 评估「加权位次」中近年权重系数（建议初版 近1年:近2年:近3年 = 0.5:0.3:0.2）。

**来源与时效**：本参数为 skill 内置默认逻辑，非官方数据。涉及的录取位次、招生计划须运行时按各省考试院官方与阳光高考（https://gaokao.chsi.com.cn/sch/）当年/近年数据核验。采集时务必标注年份与采集时间。
