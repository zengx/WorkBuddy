# T1 · 专业初筛表（**已与新模板结构脱节，仅作降级蓝本保留**）

> ⚠️ **重要：本文件描述的是旧的"双子表勾选 + 优先级排序"形态，与当前公开模板（**单子表「专业初选」**，7 列 = 门类 / 专业类 / 一句话介绍 / 是否考虑？（填是/否）/ 该专业类吸引我的原因？/ 要注意的专业 / 需要了解的专业形势）**不一致**。
> **主路径不使用本文件**：阶段二建 T1 走 `manage.copy_file` 直接克隆公开模板（file_id 见 `references/templates.md`），版式与预填列随副本继承，AI 不需要按本模板逐格 `set_range_value` 重建。
> **本文件保留两个用途**：① 历史"双子表"机制的人类可读说明（仅文档考古价值）；② **降级蓝本**——当 `copy_file` 完全失败时，回退到 `manage.create_file(file_type="sheet", title=...)` + `sheet.set_range_value` 按本模板重建表头与初始行（建表后先 `sheet.get_sheet_info(file_id)` 取真实 `sheet_id`），**且降级版结构会与新模板不一致，须在用户话术中告知体验降级**。
> 副本/重建表的 `file_id` 须存入会话状态，用户回复"我填完了"后用 `get_content` 读取。
